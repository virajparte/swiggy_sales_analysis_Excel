# swiggy_sales_analysis_Excel

## 📌 Project Overview
This project is an interactive Power BI dashboard created to analyze Swiggy sales performance across different states, food categories, and time periods.  
The dashboard helps in understanding customer behavior, sales trends, ratings, and order performance using visual analytics.

---

## 📊 Dashboard Highlights

### 🔹 KPIs
- Total Sales: ₹53.01M
- Average Rating: 4.34
- Rating Count: 5.59M
- Total Orders: 197.43K
- Average Order Value: ₹268.51

---

## 📈 Features Included

### ✅ Monthly Trend Analysis
Tracks sales performance month-wise.

### ✅ Daily Trend Analysis
Analyzes sales across weekdays.

### ✅ Weekly Trend Analysis
Shows weekly sales consistency and fluctuations.

### ✅ Quarterly Trend
Compares quarterly:
- Sales
- Ratings
- Orders

### ✅ Veg vs Non-Veg Analysis
Displays contribution of Veg and Non-Veg food categories.

### ✅ State-wise Sales Map
Visual representation of top-performing states in India.

### ✅ Top 5 Sales States
Highlights highest revenue-generating states.

### ✅ Interactive Filters
Users can filter dashboard by:
- Month
- Location

## 📌 Business Insights
- Non-Veg food contributes more sales than Veg items.
- Karnataka recorded the highest sales.
- Customer ratings remained consistently high.
- Weekend sales are slightly higher than weekdays.
- Sales remained stable across weeks.
